IO:WALRead

WALRead is a PostgreSQL wait event that occurs when a process is waiting to read WAL (Write-Ahead Log) data from disk. This typically happens during crash recovery, standby replay, or logical decoding, when the required WAL records are not already in memory (e.g., not in the WAL read buffer or WAL receiverâ€™s buffer) and must be fetched from archived WAL files or the pg_wal directory.
